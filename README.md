# RealEnglish - Crossword Adventure

Welcome to the **RealEnglish** repository! 

RealEnglish is a unique, AI-driven educational game that transforms English vocabulary learning into an engaging crossword puzzle experience. Instead of relying on static word lists, RealEnglish features a dynamic generation engine to provide infinite, context-based crossword puzzles for users.

## 🚀 Features

- **Dynamic Crosswords:** Procedurally generated puzzle boards so you never play the same game twice.
- **Offline Starter Pack:** Includes 30 pre-loaded words to start playing and learning immediately without an internet connection.
- **Local AI Integration:** Connects to local AI models (via Ollama) to dynamically generate new English vocabulary and contextual clues tailored to your learning level (A1, A2, B1, etc.).
- **Vocabulary Studio:** Build your own custom dictionary. Save learned words, track your repetition count, and manage your language progress using our smart algorithms.
- **XP & Streaks:** Gamified progression system to keep you motivated.

## 🎨 Premium Design
The application features a fully custom, handcrafted "Cyberpunk / Neon" user interface, completely breaking away from generic templates to offer users a visually stunning, premium experience.
